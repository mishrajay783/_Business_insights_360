Sales Insights 360

Project Description:
After completing the Codebasics Power BI course, I developed a clearer understanding of problem-solving approaches that extend beyond tool usage. The project strengthened my skills in SQL, Power BI Desktop, Excel, the DAX language, and DAX Studio for building optimized and reliable analytical reports.

Key Power BI Techniques Applied:
✔ Developed calculated columns and measures using DAX.
✔ Designed a well-structured and efficient data model.
✔ Utilized Bookmarks and interactive buttons for smooth visual navigation.
✔ Used the DIVIDE function to ensure error-free calculations.
✔ Created a dynamic date table using M language.
✔ Implemented dynamic report titles based on active filters.
✔ Added KPI indicators with icons and conditional formatting.
✔ Performed data validation to maintain data accuracy.
✔ Explored Power BI Service features such as report publishing, personal gateway setup, and app creation.
✔ Gained experience in workspace management, collaboration, and access-level permissions.

Important Business Metrics Covered:
✔ Analyzed Gross Price, Pre-Invoice Deductions, Post-Invoice Deductions, Net Invoice Sales, Cost of Goods Sold (COGS), Gross Margin, Net Sales, and Net Profit.
✔ Utilized YTD (Year-to-Date) and YTG (Year-to-Go) metrics for time-based insights.
✔ Compared sales performance across Direct customers, Retailers, Distributors, and Consumers.
